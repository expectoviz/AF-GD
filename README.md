# AF (Admin Finder)
This tools will help anyone to find login admin page with effective wordlist

Installation

- apt-get install git
- apt-get install php
- git clone https://github.com/expectoviz/af.git

How to run?

- cd af
- php af.php
- input target
- input list file : wlist.txt or your own wordlist
